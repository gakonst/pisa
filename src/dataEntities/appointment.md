## challengePeriod

### Challenge period

The number of blocks that PISA has to respond if an event is noticed

`challengePeriod`

- is **required**
- type: `integer`
- defined in this schema

### challengePeriod Type

`integer`

- minimum value: `0`
- maximum value: `9007199254740991`

### challengePeriod Example

```json
100
```
